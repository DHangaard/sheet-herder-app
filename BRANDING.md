# Branding and Identity

This document outlines the guidelines for the visual identity, name, and logo of this project. It clarifies which elements are protected and are not covered by the project's open-source license.

## Purpose
The goal of this project is to serve as a professional portfolio piece. While the source code is open for everyone to learn from and build upon, the unique brand identity is maintained to ensure the project remains recognizable as the work of the author.

## Protected Elements
The following elements are **not licensed for reuse** and are protected by copyright:

- **Project Name:** The specific name of this application.
- **Logo:** The visual mark in all its forms.

### Logo Variants
The protection of the logo extends to all versions and variations of the visual mark, including but not limited to:
- Full-color versions.
- Monochromatic (black/white) versions.
- Alternative color palettes used for dark/light mode themes.
- Simplified versions, such as favicons or app icons.

## Visual Identity
Other than the project name and logo, the visual presentation of this project is not intentionally restricted.

Layout, UI patterns, typography choices, and general styling may be reused, adapted, or modified in other projects unless explicitly stated otherwise.

## Third-party Assets
This project uses third-party assets such as open-source icon libraries (e.g., Iconify) and publicly available color palettes (e.g., Coolors).

These assets are licensed by their respective authors and are used in accordance with their licenses.

## Usage Guidelines

### You are encouraged to:
- View and interact with the branding assets within this repository.
- Reference the project name and display the logo for educational purposes, reviews, or when featuring this project in a portfolio context.
- Link back to this repository.

### You may not:
- Use the project name or logo for your own software projects.
- Create derivative works based on the logo (e.g., modifying the shape or colors for a new brand).
- Redistribute the branding assets as part of a commercial product.
